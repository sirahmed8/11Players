# 11Players

11Players is a gamified football matchmaking and community management web application. It allows players to create profiles, set their physical attributes, choose their positions, and generate dynamic "Player Cards" akin to popular football video games (like eFootball or FIFA).

## Features

- **Player Onboarding**: Create a detailed football profile including stats, play styles, and positions.
- **Dynamic Player Cards**: Generate a visual representation of your skills.
- **Matchmaking & Balancing**: Algorithms to balance teams based on player skills and preferred positions.
- **Admin Console**: Manage players and community guidelines.
- **Bilingual Support**: Fully supports English and Arabic (RTL).

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Firebase (Auth, Firestore, Storage)
- Framer Motion
