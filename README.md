# 11Players

![11Players Cover](/public/logo.jpg)

**11Players** is the premier platform for managing football (soccer) communities. It allows groups to create their own communities, add players, track stats, and automatically organize highly-balanced matches using a custom matchmaking algorithm based on real-life attributes and positions.

## 🌟 Key Features

- **Player Management**: Create and manage detailed player profiles including positions, attributes, and special skills.
- **Fair Matchmaking**: Advanced algorithm to divide players into perfectly balanced teams based on Position Specific Index (PSI) and overall ratings.
- **Peer Reviews**: Player attributes are rated by peers to ensure accuracy and prevent inflated stats.
- **Stat Tracking**: Keep track of goals, assists, matches played, and MVP awards across multiple communities.
- **Season Trophies**: End the season to distribute permanent profile trophies (Ballon d'Or, Golden Boot, Playmaker).
- **Progressive Web App**: Install the platform on your home screen for a native app experience.

## 🚀 Technologies Used

- **Frontend**: Next.js 14, React, Tailwind CSS, Framer Motion
- **Backend**: Firebase (Firestore, Authentication, Storage)
- **Deployment**: Firebase Hosting
